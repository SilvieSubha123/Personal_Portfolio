# Personal Profile Webpage

This is a simple personal profile webpage created using basic HTML.

## About
The webpage contains a short introduction, educational background, and hobbies of **Subhajit Mishra**, a B.Tech (CSE) 3rd-year student.

## Technologies Used

## Purpose
This project is made to practice HTML fundamentals such as headings, paragraphs, line breaks, and page structure.  
It also serves as a base for a future personal portfolio.

## How to Run
1. Download or clone the repository
2. Open `index.html` in any web browser
